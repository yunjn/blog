Markdown

2020-06-01- 2020-11-15
头秃-note

工具：vs code
插件:Markdown Markdown pdf Markdown preview enhanced Markdown formatter Markdownlint
PS:其实也可以只装个Markdown All in One

标题

# 一级标题 ## 二级标题 直到六级标题

字体

斜体 *斜体* 粗体 **粗体** 粗斜体 ***粗斜体*** 删除线 ~~删除线~~

段落引用

> + 文字(一级)

>> + 文字(二级)

>>> + 文字(三级)

分割线

  • 使用三个或以上的 *和_都可

图片

  • 行内式语法

  • ![图片名称](图片地址 “图片提示”)

  • 图片名称就是对图片内容的解释，如果图片因为某些原因不能显示， 就用定义的图片名称文字来代替图片。图片提示是当鼠标悬停在图片上时显示的提示内容。图片提示可加可不加

  • 链接本地图片:![](地址\filename.etc)

超链接

行内式

  • [超链接名称](超链接地址 “超链接提示”)
  • 例 : GitHub

自动链接

  • <链接>如 https://zhuqin.top

参考式

  • [名称][代号]

  • [代号]:地址

列表

  1. 无序列表用 - + *任何一种开头都可以。注意- + *跟内容之间要有一个空格
  2. 有序列表用数字加点为开头。注意序号跟内容之间要有一个空格
  3. 列表嵌套在VS Code里用Tab键就能使用列表嵌套的功能，上一级与下一级的序号之间相差2个空格(也有配置成3个空格的)

例

  • 列表1
    • 二级列表1
      • 三级列表1
  • 列表2
    • 二级列表2
      • 三级列表2
  • 列表3
  1. 列表1
    1. 二级列表1
      1. 三级列表1
  2. 列表2

表格

  • 每一行用|分割列。第二行分割表头和内容，可以在这里控制该列的对齐方式。左边有 ‘:’ 表示左对齐，右边有 ‘:’ 表示右对齐，两边都有 ‘:’ 表示居中对齐，中间的 ‘-‘ 是为了占满空间

例

左对齐 右对齐 居中对齐
1 2 3
a b c

代码

  1. 单行代码 : 代码之间分别用一个反引号( ` )括起来
    • #include<iostream>
  2. 代码块 : 代码前后各一行用三个反引号( ``` )括起来第一个```后写上代码类型
#include<iostream>
int main()
{
    std::cout<<"hello markdown";
    return 0;
}

脚注

  • 待解释[^name]
    然后[^name]:内容

列 :待解释^1

任务列表

* [x] 内容
* [] 内容
   x表示已完成

锚点

  • 例: 跳转到标题
<a id="01">1.标题</a>

 跳转到[1.标题](#01)

Todo列表

  • 123
  • 456
使用带有 [ ] 或 [x] （未完成或已完成）项的列表语法撰写一个待办事宜列表，并且支持子列表嵌套以及混用Markdown语法
* [ ] 123
* [x] 456

注意

  • 换行需要两个空格加回车
  • &emsp;表示缩进一个中文字符
  • &ensp;表示缩进半个中文字符
  • &nbsp;表示缩进1/4个中文字符
  • 全文使用的无序列表的序号应当一致
  • 代码区```后写明语言
  • 全文统一无序列表符
  • \表示转义字符防止[（等打印不出）
